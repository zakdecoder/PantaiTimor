About This Project

The SLMS is a self-built web-based application that allows administrators to manage course profiles, student profiles, and course-student enrolment relationships. Built with HTML, CSS, JavaScript (frontend) and PHP (backend) as part of CSEB5223 Software Construction & Methods coursework at Universiti Tenaga Nasional (UNITEN).
